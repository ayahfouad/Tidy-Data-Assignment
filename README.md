# Tidy-Data-Assignment
Peer-graded Assignment: Getting and Cleaning Data Course Project


##### This Repository is to submit the thid course's project in Coursera "Data Science Specialization", which is graded by peers as part of "Peer-graded Assignment: Getting and Cleaning Data Course Project" section.

## Data Source
http://archive.ics.uci.edu/ml/datasets/Human+Activity+Recognition+Using+Smartphones

## Files
- **Code_Book.md** : which is a code book that describes the variables, the data, and any transformations or work that I performed to clean up the data
- **run_analysis.R** : which performs:
  1. The data preparation
  2. Merges the training and the test sets to create one data set.
  3. Extracts only the measurements on the mean and standard deviation for each measurement.
  4. Uses descriptive activity names to name the activities in the data set.
  5. Appropriately labels the data set with descriptive variable names.
  6. Creates a second, independent tidy data set with the average of each variable for each activity and each subject
  
 - **FinalData.txt** : which is the exported final data after going through all the steps described above.
